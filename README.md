## Adding Custom Commands
To add this custom commands go to [Control panel](https://yagpdb.xyz/manage), login and select your server. Than go to core -> custom command. Now create custom commands and read heading comments in code to set that up.
> If you're new, read [The custom command interface](https://learn.yagpdb.xyz/the-custom-command-interface)
