## About

Objective is to be the first to match 4 same coloured disc. Player 1 will get the red disc and player 2 will get yellow disc. See the rules by using the command: `-connect4 help`

Features:\
• Play the game by adding reaction\
• Highlights player turn and move\
• Get redirect link of an existing game\
• You can play by yourself

## Installing

These commands aren't standalone. You need add both of these to have a working command.

**Installing main.go.tmpl**\
• Open the main file\
• Copy the whole code\
• Go to [control panel](https://yagpdb.xyz/manage/) -> core -> custom commands\
• Create a custom command\
• Paste the code in response\
• Now to set up the trigger, read comments at the top of the code\
• Save and Exit :)

**Installing reaction_listener.go.tmpl**\
• Open the reaction_listener file\
• Do the same steps as you did earlier
