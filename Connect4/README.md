## About {#1}
This is a Connect4 game. In this game, your object is to be the first to match 4 same coloured disc. Player 1 will get the red disc and player 2 will get yellow disc. See the rules by using the command: `-connect4 help`

Features:
• Play the game by adding reaction
• Highlights turn and move of player
• Announces the winner as soon as possible
• Get redirect link of an existing game
• Can be played by yourself

## Installing {#2}

Installing main.go.tmpl
: Open the main file
: Copy the whole code
: Go to [control panel](https://yagpdb.xyz/manage/) -> core -> custom commands
: Create a custom command
: Paste the code in response
: Now to set up the trigger, read comments at the top of the code
: Save and Exit :)

Installing reaction_listener.go.tmpl
: Open the reaction_listener file
: Do the same steps as you did earlier
